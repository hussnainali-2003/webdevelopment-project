# Assignment 2 — Responsive Top Menu with Hamburger

Adds a Tesla-style fixed top navigation bar with:
- Logo on the left
- Centre vehicle links (desktop)
- Right account links (desktop)
- Hamburger icon (mobile, ≤768px) → opens slide-in side panel with overlay
- Close button + outside-click + Escape-on-resize all close the panel

## Stack
HTML5 + CSS + Vanilla JavaScript only — no framework, no jQuery.

## Run
Open `index.html`. Resize browser below 768px width to see the hamburger appear.
