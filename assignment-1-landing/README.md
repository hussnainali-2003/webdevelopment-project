# Assignment 1 — Tesla Landing Page (HTML5 + CSS only)

Pure HTML5 + CSS clone of tesla.com's homepage.

## Run
Just open `index.html` in your browser. No build step.

## Features
- Semantic HTML5 (`header`, `nav`, `main`, `section`, `footer`)
- Single external `style.css` — no frameworks, no JS
- 4 full-bleed vehicle sections (Model Y, 3, S, X) with overlay headline + CTA buttons
- Tesla-style top nav and minimal footer

## Files
- `index.html` — markup
- `style.css` — all styling
- `public/images/` — 4 hero images
