# Tesla Landing Page

## How to run
1. Install Node.js
2. Open terminal in this folder
3. Run `npm install`
4. Run `npm start`
5. Open http://localhost:3000

## Features
- Responsive design (mobile & desktop)
- Hamburger menu for mobile
- Product carousel with infinite loop, responsive breakpoints, prev/next, slide counter, autoplay (5s), pause on hover
- Express server
