# Lab Task 1 — Responsive Tesla Landing

Builds on Assignment 1 by adding responsive `@media` queries.

## Breakpoints
- `max-width: 768px` — mobile/tablet: hides centre nav, stacks CTA buttons, single-column footer
- `max-width: 480px` — phone: shrinks logo and headlines

## Run
Open `index.html`. Resize the browser window or use DevTools device toolbar to test mobile view.
